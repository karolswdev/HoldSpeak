# HoldSpeak - Future Ideas

## Current Features (v0.1)
- Hold-to-record voice typing with configurable hotkey
- Local transcription via MLX Whisper (Apple Silicon optimized)
- Beautiful TUI with real-time status indicators
- Audio level meter during recording
- Transcription history with click-to-copy
- Settings modal for hotkey and model selection
- Auto-paste into active application
