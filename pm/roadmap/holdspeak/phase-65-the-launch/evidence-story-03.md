## The demo GIF: the honest fallback (taken)

The brief allowed a README demo GIF "gated on quality, honest fallback:
skip and say so." Taken, and here is the reasoning, not a hand-wave: the
hero moment is "speak, and watch it type into your editor" — that needs
real microphone audio, a real target application, and a screen recording
of synthetic typing landing in it. A headless capture of the web UI is
not that moment, and a staged or sped-up fake would violate the project's
own honesty bar (the same bar that killed inflated learning counts and
privacy novels). So no new GIF this phase.

What carries the visual story instead, already in the README and all real:
the `/history` artifact screenshot, the dictation journal, the learning
digest, and Qlippy's native overlay on a real Linux desktop, plus the
existing pixel-art operator-loop GIF. A true speak-and-type screencast is
a one-take manual capture the owner can drop in later; it is noted in the
kit's posting notes.
