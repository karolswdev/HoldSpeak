## Problem

Four launcher idioms (room menu, dock, tool shelf, start actions)
teach four ways to do one thing (UIUX_JUDGMENT §5.4). Thesis §2.

## Scope

- In: the dock carries the four applications + running marks + the
  record orb and nothing else; the menu bar slims to system truth
  (HoldSpeak menu = the four apps + the desk, trust badge, clock);
  the tool shelf becomes ⌘K search (apps, tools, desk objects, verbs)
  with tools/Workbench/Components reachable through it; start actions
  remain only as the empty-desk arrival verbs.
- Out: removing capabilities the shelf reached (search must reach them).

## Acceptance criteria

- [ ] One launcher idiom + ⌘K; the old shelf/room-menu chrome is gone.
- [ ] The chrome walk leg extended: search reaches every app and tool.
- [ ] Live screenshots at 1440 and 393.
