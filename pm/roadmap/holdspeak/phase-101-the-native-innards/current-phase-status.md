## Why this phase exists

Two fronts, one phase, both from the owner's close-of-100 words:

1. **Native innards** (AGENT_BRIEF.md §5): Phase 100 fixed the SHAPE
   of every interior; the bodies still COMPOSE as generic kit —
   label-over-input stacks, uniform SurfaceRows, one text size, verbs
   in action rows. Six commitments: an interior type scale; data as
   material (edit in place, kill the form stacks); purpose-built
   compositions (the journal reads like a journal, Blocks like a
   library, Runs on like a switchboard, the transcript like a
   script); verbs on the material; direct manipulation; motion as
   meaning.
2. **OS territory, unabashed** (AGENT_BRIEF.md §6): the system shade
   behind the bell, the global keyboard grammar (⌘1–4, ⌘W/⌘M, ⌘/),
   universal right-click, drag-and-drop as a system verb (files onto
   the desk import; objects drag through the glass both ways), system
   moments, Files-grade browsing. The bar: desktop reflexes WORK here.
