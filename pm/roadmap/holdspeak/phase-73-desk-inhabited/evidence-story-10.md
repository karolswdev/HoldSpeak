## The walk

ONE continuous Playwright session, one seeded hub (a meeting with real
persisted intel + a derived artifact, an agent, a fresh desk otherwise),
`location.pathname` asserted `/` after EVERY beat:

1. **arrive** — the world, the chrome, the orb.
2. **create+edit note** — `+ Note` → the beat → typed title/body in the
   in-world editor → Escape settles.
3. **arrange** — a real drag to a deliberate spot.
4. **zone+file** — `+ Zone` arrives renaming → "Today" → the note dragged
   onto the tray → filed via the real membership PUT.
5. **dive+open+edit+surface** — the camera dive → the note's pull-out →
   Edit swaps the inline editor → Escape → `← All`.
6. **meeting drawer+stack** — Summary/Actions/Artifacts → the artifact in
   place → ← back → Escape.
7. **rail run (real model)** — the ask went through the hub's engine to
   the `.43` endpoint; the REAL model answered `"inhabited"` to "Say the
   word 'inhabited' and nothing else." (instruction followed — the
   prompt demonstrably reached it).
8. **orb live+settle** — a real `runtime_activity` broadcast flipped the
   orb; an idle frame settled it.

**8/8 beats, pathname never left `/`, zero page errors.**
